Easy way to check your daily money with IFTTT

Я ненавижу финансовое планирование, но хочу, чтобы мои ежедневные траты были более равномерными и предсказуемыми.

В любой момент времени я знаю количество моих свободных денег и количество дней, оставшихся до получения заработной платы. Этой информации достаточно, чтобы примерно рассчитать, сколько денег я могу тратить ежедневно.

Мне не хотелось заводить какой-то отдельный инструмент для такого нехитрого учёта финансов, поэтому я написал небольшое приложение, которое интегрируется с замечательным сервисом IFTTT, взаимодействовать с которым умеет множество привычных приложений, например Telegram.

Выглядит это так:

@me: #ihave 10000
@ifttt: Wait a minutes
@ifttt: 588 a day

Инструкция:

  1. Зарегистрироваться в IFTTT
  2. Узнать свой уникальный ключ на странице настроек. Выглядит ПРиМерНО_ТаК
  3. Создать новый апплет и в секции That выбрать
    Webhooks / Make a web request
  4. В поле URL указать
    https://ifttt-daily-money.herokuapp.com/calculate/КЛЮЧ/ДЕНЬ/{{Text}} ,
    подставив в адрес ключ из второго шага и число, когда приходит зарплата
  5. Создать ещё один апплет, в секции This выбрать
    Webhooks / Receive a web request
  6. Заполнить поля
    Event name: pocketMoney
    Message text: {{Value1}}
  7. Готово! Теперь первый апплет будет отправлять запрос в приложение, а второй принимать его ответ
